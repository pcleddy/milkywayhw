Navigation

ScrollZoom in / out (two-finger up/down or mouse wheel)
Swipe L/RStrafe horizontally (two-finger left/right)
DragLook around (click + drag)
W / SMove up / down (relative to view)
A / DOrbit: rotate left / right
↑ ↓Orbit: tilt up / down (elevation)
← →Orbit: rotate left / right
Q / EMove target up / down off the plane

Shortcuts

TStart / stop guided tour
HFly home to Sol (our Sun)
GGalaxy overview — overhead, 50% visible
1Mode: All (stars + planet hosts)
2Mode: Planets only (exoplanet hosts)
3Mode: Stars only (no planet hosts)
LToggle layer panel (inc. sector grid overlay)
XToggle coordinate grid (lat/long lines)
ZToggle sector grid (named sectors)
?Toggle this help screen
ESCClose overlays / end tour

Interaction

ClickSelect a star — see info + portrait panel
Dbl-ClickRe-center navigation on that point in space
Planet btnOpen surface sky view from that planet
