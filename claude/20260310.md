## Architecture

All three maps share a common pattern:

- **Renderer:** Three.js WebGLRenderer with antialiasing, pixel-ratio capped at 2
- **Camera:** PerspectiveCamera with orbit controls (yaw/pitch/distance around a target point)
- **Navigation:** Smooth interpolation (lerp 0.12) for orbit, zoom, and fly-to transitions
- **Orbit camera:** Full 360° rotation — no pitch clamping. Camera up-vector flips at poles for seamless over-the-top orbiting. Yaw drag direction reverses when upside-down for natural feel.
- **Zoom:** Scroll wheel with dolly-through when past minimum distance (pushes target forward instead of just reducing orbit radius)

## Milky Way Map (milkyway.html)

### Stars
- 135 notable stars with galactic coordinates (dist from center in ly, angle in radians, height above plane in kly)
- Scale: 0.001 (1 scene unit = 1000 light-years)
- Sun positioned at dist=26,000 ly, angle=1.3 rad from galactic center
- Star class stores: name, type (spectral), dist, desc, etym, constellation, distFromSun_ly, planets
- Color-coded by spectral type; size scaled by luminosity class

### Constellations
9 constellations with line-connected stars and labeled centroids:

| Constellation | Stars | Angular Spread |
|---------------|-------|----------------|
| Orion | 7 | 18.4° |
| Gemini | 7 | 22.2° |
| Big Dipper | 7 | 30.2° |
| Cassiopeia | 5 | 11.7° |
| Cygnus | 5 | 14.4° |
| Scorpius | 6 | 28.4° |
| Southern Cross | 4 | 8.0° |
| Taurus | 5 | 14.4° |
| Canis Major | 5 | 11.7° |

- Buttons arranged in a 3×3 CSS grid
- Clicking a constellation button flies camera to view it from Sol's perspective
- Constellation labels fade with distance (opacity based on camera proximity to centroid)

### Star Info Popup
- Photo-viewer panel shows: star name, etymology (culture/era/translation), spectral type, description, distance from Sol, sector
- 70 of 135 stars have etymology data (all constellation stars covered)
- Etymology sources: Arabic (~900 CE, dominant), Greek (~300 BCE–200 CE), Latin, Sumerian (~3000 BCE), Tupi/Brazilian, Modern
- 19 stars have planet data with clickable planet buttons
- "← Back to [star]" button restores star view after viewing planet details

### Other Features
- Minimap (top-right corner) showing overhead galaxy view with camera position indicator
- Layer toggles: constellation lines, labels, galactic grid
- HUD with camera coordinates
- Help overlay with controls reference

## Solar System Map (solarsystem.html)

- 38 bodies: Sun + 8 planets + dwarf planets + major moons
- Orbit paths rendered as line loops
- Guided tour mode stepping through bodies with descriptions
- Mode switcher (realistic vs. schematic scale)
- Body info popup with mass, period, and sky-visibility notes

## Galaxy Explorer (galaxy-explorer.html)

- 123 galaxies from the local universe
- Hubble classification types (spiral, elliptical, irregular, lenticular)
- Distance and redshift data
- RA/Dec grid overlay option
- Megaparsec grid overlay

## Mobile / Touch Support (all three files)

- `touch-action: none` on canvas to prevent browser gestures
- One-finger drag = orbit (yaw + pitch)
- Two-finger pinch = zoom
- Single tap = select object (300ms delayed to distinguish from double-tap)
- Double tap = fly to object
- Responsive CSS `@media (max-width: 768px)` adjusting HUD, controls, minimap, photo-viewer sizing
- Viewport meta tag on all files

- JS syntax validation (extracts `<script>` content, runs through `new Function()`)

- Etymology coverage (all constellation stars must have etym)
- Star class required properties (name, type, dist, desc, etym, constellation, distFromSun_ly, planets)
- Planet popup navigation (back button existence, restoration, cleanup, hiding)
- DOM element ID references (all getElementById calls resolve)
- Touch support (touch-action CSS, touchstart/move/end handlers)
- Mobile CSS media queries
- Viewport meta tags
- Cross-file consistency (Three.js loaded, file size limits, console.log audit)
